# Telecom Churn Prediction Pipeline

This repository contains a professional automated pipeline to evaluate machine learning models for predicting telecom customer churn.

## Project Overview
The goal of this project is to streamline the evaluation of different classification models. It handles data loading, preprocessing, and model comparison using Stratified K-Fold Cross-Validation.
